Project Title: FearLess

Tagline: Virtual exposure but real confidence

Description

The VR Fear Simulation System is an interactive virtual reality application designed to help individuals gradually overcome common fears through controlled exposure. The system uses immersive 3D environments to simulate real-life situations that may trigger fear, allowing users to experience and confront these scenarios in a safe and controlled setting.

The project is based on the concept of Exposure Therapy, a psychological technique used to reduce fear and anxiety by gradually exposing individuals to the source of their fear. By recreating these environments in virtual reality, users can face their fears step-by-step without real-world risk.

The application includes several simulated environments representing common phobias, such as fear of heights, fear of darkness, fear of public speaking (stage fear), and fear of insects. Each environment is designed to create a realistic experience using interactive 3D scenes, movement controls, and immersive visuals.

Team Name: Meta Mind
Team Members: Aksa Joy, Baby Mumthas P, Hannan Mariyam, Elizabeth Mariyam, Anagha KR, Adheena Beeny, Bhagya Ashok, Maria Christo, Fathima Rasha Abdul Rahim

Project Title: SyncArena

Tagline: Sync Together. Survive the Arena.

Description

SyncArena is a VR-based collaborative team game designed to improve teamwork, communication, and time management.

At the beginning of the level of the game, each player receives a brief interactive tutorial explaining the skills or objectives they need to achieve. Once the tutorial is completed, the team enters a level where different tasks are distributed among the members.

Players must communicate and coordinate with each other to complete all tasks within a limited time. Each level has a countdown timer, encouraging effective planning and collaboration. The team can only achieve the level's target score if all members successfully complete their assigned roles together.

To ensure balanced participation and learning, tasks rotate between players in every level, preventing any player from receiving the same task repeatedly.
